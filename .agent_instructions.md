# 링커엑스 모바일 ERP (.agent_instructions.md)

이 파일은 코딩 에이전트(Antigravity)가 링커엑스 모바일 프로젝트의 설계 가이드, 디자인 사상, 핵심 기능 히스토리를 유지하고 파악하기 위한 가이드라인 문서입니다. 모든 개발 명령 수행 전에 이 파일을 반드시 참조하고, 변경이 발생할 때마다 실시간으로 갱신해야 합니다.

---

## 1. 기술 스택 & 개발 환경
- **프론트엔드**: React, Vite, Tailwind CSS (index.html에서 Play CDN 로드 및 테마 연동)
- **아이콘 라이브러리**: `lucide-react`
- **백엔드/DB**: Firebase Firestore (실시간 연동 데이터베이스)
- **배포 플랫폼**: Vercel Production
- **PWA 설정**: `public/manifest.json` 연동, standalone 디스플레이 설정, portrait(세로) 화면 고정

---

## 2. 디자인 및 UI 사상
- **전체 테마**: 다크 글래스모피즘 (`bg-[#0b0f19]`, `bg-slate-900/90 backdrop-blur-md`, `border-slate-800/50`)
- **가시성 최우선**:
  - 사용자 입력창(`input`, `textarea`)은 명확한 가시성을 위해 반드시 **흰색 배경(`bg-white`)**과 **검은색 글자(`text-slate-900`)**로 고대비 처리합니다.
  - 회색 텍스트(예: `text-slate-500`)는 어두운 배경 위에서 묻히므로 중요 텍스트와 레이블에는 **`text-slate-300`** 이상의 밝은 텍스트를 사용합니다.
- **글로벌 폰트 크기 동기화**:
  - `loginFontSize` 상태가 변경되면 `localStorage`에 자동 저장되며, `document.documentElement.style.fontSize` 조정을 통해 애플리케이션 내의 모든 상대 단위 텍스트(`rem`, `em`, `%`)가 실시간으로 확대/축소됩니다.
- **글로벌 동일 적용 규칙 (No Tenant Branching)**:
  - 로그인 대리점/회사별(tenant)로 분기하는 조건문(예: `if (companyId === 'DMK')`)을 사용하여 메뉴 구조나 UI/기능을 차별화하지 않습니다.
  - 모든 대리점/회사 및 모든 사용자가 링커엑스 모바일의 동일한 기능, 메뉴 구성, 디자인 테마를 고스란히 공유하여 사용할 수 있도록 일관적인 글로벌 코드를 작성합니다.
  - 모바일 콘솔에서 내리는 지시는 모든 대리점 환경을 포함해 공통 적용되도록 빌드 및 배포됩니다.

---

## 3. 실시간 에이전트 제어 시스템
- **동작 방식**: 
  - 사용자가 모바일 콘솔 혹은 에이전트 통제 위젯에서 명령을 전송하면 `companies/DMK/agentCommands` (혹은 `devCommands`) 컬렉션에 `status: "pending"` 상태로 등록됩니다.
  - 백그라운드에 구동 중인 데몬(`agent-control-daemon.mjs`)이 이를 감지하여 프로세스를 강제 종료시킴으로써 에이전트를 실시간으로 기동시킵니다.
  - 에이전트는 기동과 동시에 해당 문서의 상태를 `running`으로 변경하고 진행 단계를 `progress`에 실시간 로깅한 뒤, 완료 시 `success`로 상태를 마킹하고 결과 팝업을 트리거합니다.
